DASHBOARD CURSO DE VIDA RIS ATE

Archivos:
- index.html
- style.css
- script.js
- data.json
- config.js

Uso:
1. Publicar esta carpeta en GitHub Pages.
2. Copiar la URL pública.
3. Pegarla en el Dashboard RIS Ate, archivo config.js, clave:
   curso: "URL_DEL_DASHBOARD_CURSO_DE_VIDA"

Datos incluidos:
- Población total RIS Ate: 310,356
- IPRESS: 12
- Cursos de vida:
  Periodo prenatal
  Niña / Niño
  Adolescente
  Adulto joven
  Adulto
  Adulto mayor

Nota:
Periodo prenatal usa población de gestantes esperadas.
Las atenciones prenatales quedan listas para actualizar cuando se valide el campo condición gestante.
